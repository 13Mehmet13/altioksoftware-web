# Altıok Software - Kurumsal Landing Page

Bu proje, **Altıok Software** için Vite + React + Tailwind CSS kullanılarak hazırlanmış modern ve mobil uyumlu bir landing page çalışmasıdır.

## Geliştirme Komutları

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Teknoloji

- Vite
- React
- Tailwind CSS
